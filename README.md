## Installation

Install my-project with npm

```bash
npm init
npm install express
npm install nodemon
```
    
## Tech Stack

**Server:** Node, Express


## Run Locally

Clone the project

```bash
  git clone https://github.com/damnthonyy/server-node.git
```

Go to the project directory

```bash
  cd server-node
```


Start the server

```bash
  npm start
```
